# MiniWeather

一个展示天气的小项目，刚接触swift不久，用来熟悉和练习语言的，好好感受了一下safety的特点。天气数据来源是forecast.io的API。UI比较简单，但是还挺可爱的呀。顺便练习了一下autolayout和sizeclass。    
地点就先选择了本地Vancouver，坐标是直接写上去的。等过了期末再去研究怎么获得用户位置。
